# Learning_scheme

## To run this project you will need

[Chicken Scheme compiler](https://www.call-cc.org/)

Run a Hello World

compile with Chicken Scheme Compiler

```
csc -o hellowordl helloWorld.sch
```
Execute
```
./helloworld
```
